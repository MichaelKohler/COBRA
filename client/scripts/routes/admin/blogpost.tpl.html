Erklärungen zu "Markdown"

Das Eingabefeld verwendet "Markdown", um Inhalt zu formatieren. Anschliessend finden Sie eine Erklärung der verschiedenen Formate.

Das ist ein grosser Titel
===== 		ergibt einen grossen Titel
Das ist ein mittlerer Titel
===== 		ergibt einen mittleren Titel
### Das ist ein kleiner Titel ergibt einen kleinen Titel
**das hier ist fett** ergibt fettmarkierten Text
*das hier ist kursiv* ergibt kursiven Text
[Beschreibung des Links](http://beispiellink.ch) ergibt einen Link auf beispiellink.ch mit dem Text "Beschreibung des Links"
